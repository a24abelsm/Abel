Bulma

← Volver al índice · Web oficial

Descripción

Framework ligero basado en Flexbox y Sass. Ofrece clases semánticas y componentes CSS: navbar, tiles, forms, etc. No incluye JS para interactividad.

Ventajas

  • Sintaxis clara y semántica
  • Muy fácil de empezar
  • Completamente basado en CSS (sin dependencias JS)

Inconvenientes

  • Sin componentes JS; hay que implementar la lógica o usar librerías extra
  • Menos ecosistema comparado con Bootstrap o Tailwind
