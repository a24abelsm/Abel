Tailwind CSS

← Volver al índice · Web oficial

Descripción

Framework utility-first: introduce miles de clases utilitarias para componer diseño directamente en HTML. Funciona mejor con proceso de build (PostCSS) para purgar CSS no usado.

Ventajas

  • Control fino del diseño sin escribir CSS personalizado
  • Pequeño CSS en producción tras purga
  • Ecosistema amplio (plugins, UI kits)

Inconvenientes

  • Curva de aprendizaje de mentalidad utility-first
  • Requiere pipeline de build para mejores resultados
  • HTML puede volverse verboso sin extracción de componentes
