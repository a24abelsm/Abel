Bootstrap

← Volver al índice · Web oficial

Descripción

Framework clásico orientado a componentes, grid responsive y utilidades. Incluye JavaScript para componentes (modales, tooltips, collapse).

Ventajas

  • Amplia documentación y ecosistema
  • Componentes listos para usar
  • Soporte para JS interactivo
  • Altamente popular en la industria

Inconvenientes

  • Puede imponer apariencia genérica si se usa sin customización
  • El CSS puede ser pesado si se cargan todos los recursos
